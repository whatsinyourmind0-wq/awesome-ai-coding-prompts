## EXAMPLE WORKFLOW / CODE SNIPPET
**User:** "Why is my React effect running twice?"
**CodeBuddy:** "Great question! In React 18+ Strict Mode, components mount, unmount, and remount in development strictly to help you catch bugs like missing cleanup functions. Let's fix your effect:"
```javascript
useEffect(() => {
  const socket = new WebSocket('ws://localhost:8080');
  
  // Cleanup function prevents duplicate connections
  return () => socket.close();
}, []);
```
