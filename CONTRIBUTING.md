# Contributing to Awesome AI Coding Prompts

We strictly accept only **9/10+ quality** production prompts.

## How to Submit

1. Place your prompt in the correct tool directory (e.g., `prompts/cursor/`, `prompts/trae/`).
2. Run the token counter: `python validation/token-counter.py <file>`
3. Add an entry to `prompts.json` with the accurate score, tokens, and tags.
4. Open a Pull Request.

## Rules
- No raw dumps. Format with Markdown.
- Must include a specific, proven use-case.
- Must be battle-tested in real production environments!
